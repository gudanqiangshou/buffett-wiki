# 1962年11月 巴菲特致合伙人信

## 核心要点

- 一年一度的文书工作又开始了。
- 随函还附上两份承诺书（一份给你留存，一份请寄回），请在上面注明你希望在1月1日增资或提现的金额。
- 提现款将在1月1日后立即拨付。
- 请务必对合伙协议上的签名进行公证。
- 关于1962年纳税申报所需的完整税务资料，将在1月20日前送到你手上。
- 读到这里，你有权利听我汇报一下我们今年以来的成绩了。

## 详细摘要

巴菲特合伙公司（BUFFETT PARTNERSHIP, LTD.）
810 KIEWIT PLAZA
奥马哈 31，内布拉斯加州
1962年11月1日
致1963年全体合伙人：
一年一度的文书工作又开始了。随函附上两份1963年修正合伙协议各一式两份。附有一般条款的那份请你留存，另一份单页协议请寄回给我们。
